N Queens — Place, Check, Backtrack

Real-world analogy

Seat one "diva" per row so none can see another along a column or diagonal. Pick a safe column, move to the next row. No safe column left? Back up the previous diva and re-seat her. All rows filled = a valid arrangement.

Key takeaway

One queen per row kills row conflicts for free. A column is unsafe if its c ∈ cols, r−c ∈ diag, or r+c ∈ anti — all O(1). Place → recurse → remove from the three sets to backtrack. 4×4 yields exactly 2 solutions.
