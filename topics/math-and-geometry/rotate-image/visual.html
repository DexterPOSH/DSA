Rotate Image — Transpose then Reverse

🧠
Real-world analogy

Azure Machine Learning or Azure AI Vision can rotate an image tensor in place to avoid allocating another full batch. Transpose swaps row/column axes, then reverse each row flips pixels into a 90° clockwise rotation.

1 · Transpose (diagonal swap) 2 · Reverse each row
Press "Next Step" to rotate [[1,2,3],[4,5,6],[7,8,9]] 90° clockwise, in place.
✅ Rotated 90° clockwise — [[7,4,1],[8,5,2],[9,6,3]] — all in place, O(1) space.
💡
Key Takeaway

Clockwise rotation = transpose + reverse each row. Two clean O(n²)-time passes with O(1) extra space beat memorizing the new[j][n-1-i] = old[i][j] index formula.

def rotate(matrix): n = len(matrix) # Step 1: transpose (swap across main diagonal, j > i) for i in range(n): for j in range(i + 1, n): matrix[i][j], matrix[j][i] = matrix[j][i], matrix[i][j] # Step 2: reverse each row (horizontal flip) for row in matrix: row.reverse()
