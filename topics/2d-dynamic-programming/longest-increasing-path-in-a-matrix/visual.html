Longest Increasing Path — DFS + memo

Longest strictly-increasing walk through a grid, moving only up/down/left/right.

⛰️ Analogy: uphill-only hiking

Grid ek height-map hai, har cell ki value altitude. Hiker sirf strictly upar ja sakta hai (4 padosi cells). "Is cell se aage longest uphill trail" fixed hai aur koi cycle possible nahi — to har cell ka answer ek baar nikaal ke memo me likh do. Green badge = memoized length.

Press Next. We compute dfs(r,c) = longest increasing path starting at each cell. Strict increase means no cycles, so a memo gives each cell's answer in O(1) after the first visit.

Key takeaway

Because heights strictly increase, the grid is a DAG — no visited set needed. Memoizing "longest path from this cell" turns an exponential search into O(m·n); the answer is the max memo value, here 4.
