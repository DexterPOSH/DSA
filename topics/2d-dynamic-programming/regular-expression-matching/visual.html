Problem. Does s = "aab" fully match p = "c*a*b"? → True  (c*→zero c, a*→"aa", b→"b").

Key Takeaway

Plain char / . → copy the diagonal if chars match. A * is never alone: it forks into zero of the preceding (look left two, dp[i][j-2]) OR one more (look up, dp[i-1][j], if the preceding char matches). Fill the grid in O(m·n); dp[m][n] is the verdict.
