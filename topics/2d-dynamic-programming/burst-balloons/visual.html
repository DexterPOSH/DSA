Real-world analogy

"Pehle kaunsa phodun?" puchhne pe fas jaaoge — padosi badalte rehte hain. Ulta socho: kisi interval (l, r) me poocho "is range ka last balloon kaunsa phoota?". Tab uske padosi exactly walls l aur r hote hain — problem do clean halves me toot jaati hai.

Problem. nums = [3, 1, 5, 8], padded to arr = [1, 3, 1, 5, 8, 1]. Burst all balloons for max coins. → Answer = 167.
What's happening
Press Next to fill dp[l][r] by interval length — shortest spans first.
Balloons (padded with walls = 1)
dp[l][r] = max coins from bursting all balloons strictly between l and r
Trying each k as the LAST balloon burst in (l, r)
—
current dp[l][r] walls l, r last burst k sub-intervals dp[l][k], dp[k][r]
💡
Key Takeaway

By fixing the last balloon k in each interval, its neighbours are pinned to the walls l and r, splitting the work into independent halves dp[l][k] + dp[k][r]. Fill length-by-length in O(n³); dp[0][n-1] is the answer.
