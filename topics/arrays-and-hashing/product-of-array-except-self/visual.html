Product of Array Except Self — Prefix × Suffix

Real-world analogy
An Azure Data Factory pipeline can run a forward aggregate pass and then a backward aggregate pass over ordered rows. Each output receives prefix product × suffix product, so the current row is excluded without division.

Key Takeaway

output[i] = (product of everything left of i) × (product of everything right of i). Two linear passes, a running prefix then a running suffix, give the answer in O(n) time, O(1) extra space — no division, zeros handled for free.
