Frequency Map (count[letter])
Key Takeaway: Frequency maps let you compare unordered collections in O(n) time. Walk string s once to build counts, walk string t once to decrement. If all counts reach zero, the strings are anagrams. No sorting required.
Python Implementation
def isAnagram(s: str, t: str) -> bool: # Early exit: different lengths can't be anagrams if len(s) != len(t): return False count = {} # or: defaultdict(int) # Phase 1: build frequency map from s for ch in s: count[ch] = count.get(ch, 0) + 1 # Phase 2: decrement using t for ch in t: count[ch] = count.get(ch, 0) - 1 if count[ch] < 0: return False # t has a char not in s return True # all counts balanced to zero # Complexity: Time O(n), Space O(1) — alphabet is bounded to 26 letters
