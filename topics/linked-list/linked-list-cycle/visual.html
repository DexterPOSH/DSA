Real-world analogy
Follow an Azure VNet/Route Server next-hop chain with two probes. If the path loops, the two-hop probe catches the one-hop probe; if it terminates, the fast probe reaches None.

Key takeaway

Move slow by 1 and fast by 2. If there's a cycle, the gap shrinks by one each step until slow is fast — they collide. No cycle? fast falls off the end (None). O(n) time, O(1) space — beats the hash-set approach on memory.
