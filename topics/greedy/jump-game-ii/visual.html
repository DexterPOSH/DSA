Jump Game II — BFS levels & boundaries

Each jump is a BFS level. Track the level boundary cur_end and the farthest reachable index.

🚌 Real-world analogy: bus-route levels

Index 0 se 1 jump me jahan-jahan pahunch sakte ho = level 1 zone. Us zone ke har stop se aage ki combined reach = level 2 zone. Har jump ek BFS level. Hum individual stops visit nahi karte — bas level ka right boundary aur us level ki farthest reach track karte hain.

Step 0
def jump(nums): jumps = cur_end = farthest = 0 for i in range(len(nums) - 1): # stop before last index farthest = max(farthest, i + nums[i]) if i == cur_end: # hit the level boundary jumps += 1 cur_end = farthest # advance to next level return jumps

💡 Key takeaway

Min-jumps is BFS over levels. farthest absorbs every index in the current level; when i hits cur_end, the level is exhausted — spend one jump and push the boundary to farthest. Stop before the last index to avoid an off-by-one extra jump. O(n) time, O(1) space.
