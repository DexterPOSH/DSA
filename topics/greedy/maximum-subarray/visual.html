Step 0 / 9
def max_subarray(nums): cur = best = nums[0] for x in nums[1:]: cur = max(x, cur + x) # extend or fresh-start (greedy) best = max(best, cur) # track overall max return best

💡 Key takeaway

A negative running prefix can only hurt future sums — so the moment cur would do better restarting at x alone, restart. One O(n) pass, O(1) space, and best captures any subarray that peaks mid-stream.
