📈 Azure analogy: VM Scale Set autoscale reach

Azure Monitor autoscale tracks the farthest VM Scale Set capacity reachable from rules already within range. If a capacity checkpoint is beyond reach, no earlier scale decision can get there; once reach covers the target, the deployment is feasible.
