Real-world analogy

Stack receipts, but stick a sticky-note on each one saying "cheapest amount down to here". A new note = min(this value, note below). Ask "cheapest so far?" → just read the top note. O(1).

Key takeaway

Keep a second stack in lock-step whose top is always the running minimum. Push two, pop two. getMin is just reading the top of the min stack — O(1) for every operation, at the cost of O(n) extra space.

def push(self, val): self.stack.append(val) m = val if not self.min_stack else min(val, self.min_stack[-1]) self.min_stack.append(m) # lock-step def pop(self): self.stack.pop(); self.min_stack.pop() def getMin(self): return self.min_stack[-1] # O(1)
