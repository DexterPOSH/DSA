Real-world analogy

A stack of plates: each open bracket puts a plate on top. A close bracket must match the topmost plate — the most recently opened one. Last opened, first closed. That's LIFO.

Key takeaway

Push opens, and on every close pop the top and check the type matches. Empty stack at the end → valid. One O(n) pass — order errors like "([)]" are caught because the stack remembers nesting order, which simple counting cannot.

def is_valid(s): pairs = {')':'(', ']':'[', '}':'{'} # close -> open stack = [] for ch in s: if ch in pairs: # closing bracket if not stack or stack.pop() != pairs[ch]: return False else: stack.append(ch) # opening bracket return not stack # empty == valid
