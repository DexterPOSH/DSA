Reverse Integer — peel a digit, push it, guard the 32-bit edge

Reverse the digits of x, returning 0 if the result escapes [-2³¹, 2³¹-1].

The whole algorithm
def reverse(x): INT_MAX = 2**31 - 1 # 2147483647 sign, x, res = (1 if x >= 0 else -1), abs(x), 0 while x: digit = x % 10; x //= 10 if res > INT_MAX // 10 or (res == INT_MAX // 10 and digit > 7): return 0 # pre-multiply overflow guard res = res * 10 + digit return sign * res

💡 Key takeaway

Reverse a number by repeatedly peeling the last digit (x%10) and pushing it onto res = res*10 + digit. The trick is checking overflow before the multiply — comparing against INT_MAX // 10 — so you never actually overflow a fixed-width int. O(number of digits) time, O(1) space.
