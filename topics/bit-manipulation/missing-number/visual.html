Missing Number — XOR cancels every pair, leaving the gap

Array [3, 0, 1], range [0, 3]. We XOR every index and value into one accumulator; the missing 2 survives.

The whole algorithm
def missing_number(nums): res = len(nums) # seed with n (top of range) for i, num in enumerate(nums): res ^= i ^ num # matching index/value pairs cancel return res

💡 Key takeaway

Every value that appears as both an index and a number gets XORed twice → it cancels to 0. The one number that is only an index (never a value) is XORed an odd number of times and survives. O(n) time, O(1) space, no overflow.
