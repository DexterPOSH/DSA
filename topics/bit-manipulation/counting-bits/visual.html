Real-world analogy
Azure Policy can cache how many Azure Service Bus SAS rights are enabled for every permission mask. For mask i, reuse the cached count for i >> 1 and add whether the dropped low flag i & 1 was enabled.

Key Takeaway

Each Azure permission mask count dp[i] reuses an already-computed smaller mask (dp[i>>1]) plus the dropped low flag (i&1). One O(1) lookup per mask → O(n) total, beating O(n log n) repeated popcount.
