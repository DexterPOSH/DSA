Sum of Two Integers — Azure VM vCPU carry propagation

Adding a = 7 and b = 5 with no + — just XOR, AND and left-shift, looping until the carry is gone.

🖥️ Real-world analogy

Inside an Azure VM, the virtual CPU's adder can split fixed-width addition into two bit operations. XOR gives the partial sum without carry, AND finds where carries are born, and << 1 moves each carry to the next bit column.

def get_sum(a, b): while b != 0: # while a carry still exists carry = (a & b) << 1 # both-1 bits, bumped one column left a = a ^ b # add the bits, ignoring carry b = carry # the carry becomes the next addend return a

💡 Key takeaway

An Azure VM vCPU adder splits addition into two bit operations: XOR adds without carrying, AND finds where a carry is born, and shifting moves that carry to the next column. Loop until no carry remains. O(1) for fixed-width ints — this mirrors hardware carry propagation.
