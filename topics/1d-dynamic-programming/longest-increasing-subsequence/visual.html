Longest Increasing Subsequence — Azure stream trend DP

dp[i] = length of the longest increasing subsequence that ends at index i.

📊 Real-world analogy

Azure Stream Analytics tracks the longest improving metric trend ending at each event. Each event reuses cached trend lengths from earlier smaller values and writes the best length for this endpoint.

Problem. nums = [10, 9, 2, 5, 3, 7] — length of the longest strictly increasing subsequence. → Answer: 3  (e.g. [2, 5, 7] or [2, 3, 7])

Key takeaway

For each Azure Stream Analytics event i, look back at earlier smaller values and reuse the cached dp[j] trend length, then write max(dp[j]) + 1. The answer is the largest value in the dp row — O(n²), or O(n log n) with patience sorting.
