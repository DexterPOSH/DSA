Climbing Stairs — a DP row that fills left to right

Count distinct ways to climb n = 5 steps, taking 1 or 2 steps at a time.

🪜 Real-world analogy

Step i tak pahunchne ke sirf do raaste the: i-1 se ek chhoti chhalaang, ya i-2 se badi chhalaang. Toh dp[i] = dp[i-1] + dp[i-2] — seedha Fibonacci.

Step description
Press Next → to start filling the DP table.
dp[i] = number of ways to reach step i
The whole algorithm
def climbStairs(n): dp = [0] * (n + 1) dp[0] = dp[1] = 1 # base cases for i in range(2, n + 1): dp[i] = dp[i-1] + dp[i-2] # Fibonacci recurrence return dp[n]

💡 Key takeaway

Har cell apne pichhle do cells ka jod hai. Bade sawaal ko chhote, already-solved sawaalon par tikaana — yahi 1-D DP ka dil hai. O(n) time, O(1) space.
