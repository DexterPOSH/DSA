Word Break — Azure config-token reachability

An Azure config boundary is reachable if an earlier cached boundary jumps to it via a known token.

⚙️ Real-world analogy

Azure App Configuration validates a connection string by splitting it into known tokens. A boundary is reachable only if a previous cached boundary is reachable and the current slice is in the token catalog.
