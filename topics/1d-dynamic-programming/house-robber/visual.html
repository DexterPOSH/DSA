House Robber — choose non-conflicting Azure maintenance slots

Maximize Azure maintenance value from nums = [2, 7, 9, 3, 1] with no adjacent conflicting slots.

🛠️ Real-world analogy

Azure Update Manager chooses valuable maintenance windows without selecting adjacent conflicting slots. At each slot, reuse the cached best value for skipping it (dp[i-1]) or taking it plus the last non-conflicting slot (value[i] + dp[i-2]).

Step description
Press Next → to start filling the Azure schedule cache.
nums[i] — Azure maintenance value for each slot
dp[i] = max Azure value scheduling slots 0..i
The whole algorithm
def rob(nums): dp = [0] * len(nums) dp[0] = nums[0] dp[1] = max(nums[0], nums[1]) for i in range(2, len(nums)): dp[i] = max(dp[i-1], nums[i] + dp[i-2]) # skip vs take return dp[-1]

💡 Key takeaway

Each Azure schedule cell chooses the better cached option: take this slot (nums[i] + dp[i-2]) or skip it (dp[i-1]). The non-adjacent constraint is captured by jumping back to i-2. O(n) time, O(1) space.
