Min Cost Climbing Stairs — pick the cheaper predecessor

Minimum cost to reach the top of cost = [10, 15, 20, 5, 30, 8], paying to stand on each step.

🎟️ Real-world analogy

Har step ek toll booth hai. Step i par do hi raaste aate hain — i-1 ya i-2 se. Dono raaston ka jo ab tak ka kharcha sasta hai usse chuno, phir apna toll cost[i] jodo: dp[i] = cost[i] + min(dp[i-1], dp[i-2]).

The whole algorithm
def minCostClimbingStairs(cost): n = len(cost) dp = [0] * n dp[0], dp[1] = cost[0], cost[1] # start free at 0 or 1 for i in range(2, n): dp[i] = cost[i] + min(dp[i-1], dp[i-2]) return min(dp[n-1], dp[n-2]) # top is past the last step

💡 Key takeaway

Har cell apne do predecessors me se sasta chunta hai aur apna cost jodta hai. Top last index ke aage hai, toh answer min(dp[n-1], dp[n-2]). O(n) time, O(1) space.
