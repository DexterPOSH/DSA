Search a 2D Matrix — binary search on a flattened grid

Row-major sorted matrix = ek lambi sorted list. Index ko on-the-fly row/col me decode karo.

📚 Analogy: serial-numbered library shelf

Saari rows jod do to ek fully-sorted list ban jaati hai. Books strictly serial order me, bas alag racks (rows) me. Ek hi binary search — bas flat index ko row = mid // n, col = mid % n se decode karo.
