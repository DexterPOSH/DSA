Koko Eating Bananas — binary search on the answer

Array me kuch nahi dhoond rahe — speed k dhoond rahe. Slowest feasible speed.

🚿 Analogy: shower temperature knob

Dheere (low k) → time zyada. Tez (high k) → jaldi. Ek threshold hai: usse upar har speed feasible (kaam in time), neeche har speed infeasible. Monotonic flip F F F T T T — binary search ke liye perfect. Hum smallest feasible dhoond rahe.

Problem. piles = [3, 6, 7, 11], h = 8 hours. Find minimum eating speed k. → answer 4.

Candidate speeds — search space is the answer range [1 … max(piles)=11]

Feasibility check for the current mid speed

Key takeaway

Jab "minimum/maximum value such that condition holds" + monotonic ho → binary search on the answer range, ek feasible(k) checker ke saath. O(n·log(max)).
