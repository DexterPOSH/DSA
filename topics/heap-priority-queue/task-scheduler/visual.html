Task Scheduler — Azure Batch max-heap with cooldown queue

At each tick, run the most frequent available task; tasks in cooldown wait in a queue.

⚙️ Analogy: Azure Batch with cooldown

Azure Batch can dispatch the task type with the largest remaining backlog while a cooldown queue protects repeated work on the same constrained resource. The max-heap picks the most-pending available task; the queue returns it only when its cooldown time arrives.
